# OI-Useful-Softwares
OI is the abbreviation of *Olympiad in Informatics*, referring to "information Olympiad". It is a discipline competition widely carried out in middle school students, and has the same physical and mathematical competition characteristics.

This repository includes some softwares on OI, they are useful and Open Source.

# 信奥常用(开源)软件
OI是*信息学*的缩写，简称“信息奥林匹克”。它是中学生广泛开展的学科竞赛，具有和物理、数学竞赛相同的特征。

这个存储库包括一些关于OI的软件，它们是有用的和开源的。
